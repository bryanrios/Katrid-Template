Installation

There's no much to say, after downloading, just go to the target folder and clone (copy / paste) the content.

Basic Templates

  • plain-html5 (full html 5 theme examples)
  • php-templates (php basic templates)
  • django-templates (django basic templates)

Layout Structure

The basic layout structure is:

  1. #header top header
  2. #left-menu left side menu
  3. #main-content main admin content
  4. #footer the main admin footer

How to create a new plain project?

  1. Make a copy of your target platform folder.
  2. Rename the copied folder with your project name.
  3. Start editing your files.
